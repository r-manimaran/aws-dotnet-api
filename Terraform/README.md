# Creating S3 Bucket using Terraform
This repository contains Terraform code to create an S3 bucket in AWS. The code is organized into modules for better structure and reusability.


## Run the Terraform
To create the S3 bucket using Terraform, follow these steps:
1. Clone the repository to your local machine.
2. Navigate to the S3 directory: `cd S3`
3. Initialize Terraform: `terraform init`
4. Review the execution plan: `terraform plan`
5. Apply the configuration: `terraform apply`

- To Cleanup / delete the resources created using Terraform use 
`terraform destroy`
